# Hamster-Namengenerator

Browser-basierter Hamster-Namengenerator als interaktiver Einarmiger Bandit mit über 1200 echten Hamsternamen.

**Live-Demo:** https://gi007i.github.io/test/hamster.html

## Features

- Slot-Machine-Mechanik mit 3 Walzen
- Münz-Einwurf-System für authentisches Casino-Feeling
- Jackpot-Feature (10% Chance: alle 3 Namen gleich)
- Web Audio API Sound-Effekte
- Konfetti-Animation bei Jackpot
- Responsive Design (Desktop & Mobile)
- 1200 echte Hamsternamen (niedliche Kosenamen, Essens-Namen, Natur, altdeutsche Namen u.v.m.)

## Anwendung starten

Einfach [hamster.html](hamster.html) im Browser öffnen. Keine Installation erforderlich.

## Spielanleitung

1. Münze einwerfen (€-Symbol klicken)
2. Am Hebel ziehen oder Button klicken
3. Auf Jackpot hoffen (alle 3 Namen gleich)!
